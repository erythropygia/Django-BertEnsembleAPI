About Us

Cer-NLP is a natural language processing project that started in March 2023. Within the scope of this project, sentiment analysis of Turkish texts will be performed and various features of the texts will be identified. The aim of the project is to enable users to obtain faster and more accurate results by automatically analyzing Turkish texts. Thanks to the Cer-NLP API, users will be able to perform these analyzes easily and quickly. The data obtained as a result of the project will also contribute to research and applications in the field of Turkish natural language processing.

What Is BERT?

  • BERT (Bidirectional Encoder Representations from Transformers) is a highly successful artificial neural network model in natural language processing (NLP) developed by OpenAI and Google.
  • Uses transfer learning technology and is trained on large amounts of natural language data. This makes it trainable for new tasks and outperforms other NLP models.
  • The BERT model has a structure that can take into account both preceding and following words to understand word order in texts.

Frequently Asked Questions

  • What is Cer-NLP?

    Cer-NLP is a Turkish text analysis API using the BERT model. This API analyzes your texts with high speed and accuracy and delivers the results to the end user.

  • Is Cer-NLP free?

    Yes! It's free to use the API for a period of time to improve our app and discover our shortcomings.
